  <strong>
   关系
  </strong>
  正陷入困难与危险期。
  <br>
   <br>
    华盛顿智库威尔逊中心（Wilson Center）的线上研讨会，要探讨台湾的总统蔡英文开始第二个任期后美台关系展望与全球秩序下台湾的角色演变。不过，美中关系仍成为研究美台关系发展时绕不开的话题。
    <br>
     <br>
      麦艾文是奥巴马主政时期美国对华政策的主要推手。他提到，
      <span href="https://www.secretchina.com/news/gb/tag/新冠肺炎" target="_blank">
       新冠肺炎
      </span>
      疫情，让华盛顿与北京对对方的误解误判更加深化，他用“敌意与仇视”来形容美中关系现在的发展趋势，充满负能量循环。他说，虽然华盛顿也有责任，但始作俑者与主因来自北京；北京许多作为造成美中关系现在的困境与危险，也是因为中国有一派人最担忧的是，华盛顿有一个终极目标。
      <br>
       <br>
        “中国有一派人认为，美国国务卿不断提到要把中国人民和中国共产党分开，区别对待，这造成一种不论是对或错的认知，也就是美国这届政府有意要寻求政权更迭，而不只是弱化中共执政的合法性。”
        <br>
         <br>
          如何才能停止两国关系的恶性循环呢？麦艾文指出，美国对华政策有坚实基础，涉及多个层面。如果不能谨慎小心处理，将让已经脆弱的美中关系更不稳定，这其中的敏感议题包括台湾、香港、南海、东海、新疆等。
          <br/>
          <br/>
          <strong>
           北京插手香港事务  美国、台湾都在看
          </strong>
          <br/>
          <br/>
          中国第十三届全国人大三次会议21日晚间举行新闻发布会宣布，将审议俗称“香港版国安法”的《全国人民代表大会关于建立健全香港特别行政区维护国家安全的法律制度和执行机制的决定》（草案）的议案。
          <br/>
          <br/>
          北京跳过香港立法会、直接插手香港事务，外界抨击这是中南海亲手埋葬对香港“一国两制、高度自治”的承诺，担忧将导致香港再陷入混乱与撕裂。
          <br/>
          <br/>
          才卸任美国国防部亚太助理部长的薛瑞福（Randall Schriver）也在会中指出，美台关系这几年的深化，“是北京自己的对台政策的失败（造成的）”，北京该检讨，现在又在香港议题上直接干预，中南海是在毁掉香港有意义的自治。这看在台湾人民眼中，本来在台湾就没有市场的“一国两制”，会更不受欢迎，也让两岸关系更为疏远。
          <br/>
          <br/>
          参与论坛的台湾驻美代表处政治组组长赵怡翔也表达了对香港局势的担忧。
          <br/>
          <br/>
          “让人不安的是，北京对香港的作法显示，他们已经不在乎台湾人民会怎么想。但我认为，中国一向的作法是先试水温，测试国际社会底线在哪，再决定怎么做，包括在南海与涉及台湾的议题上，过去做法都是如此。所以，国际社会必须持续关注香港局势”，赵怡翔说。
          <br/>
          <br/>
          外在压力逼迫中国，白宫在20日深夜也将长达二十页的《美国对中华人民共和国的战略方针》（United States Strategic Approach to The People's Republic of China）的报告，送交国会，全面检讨华盛顿过去接触交往政策的失败，并指出美国因此决定采取“公开施压”的做法。
          <br/>
          <br/>
          麦艾文说，随着美国总统选战白热化，他认为美中对峙的态势只会更明显激烈。他说，中国必须知道，美国年底的总统大选即使是民主党的候选人拜登（Joe Biden）赢了，美中关系也好不了，“华盛顿对北京的担忧与关切在美国对华外交政策圈子里是持续的，且有两党共识。”
