 <p>
  “
  <strong>
   甩锅
  </strong>
  ”是网络流行语，意思是将所有的过错算到别人的头上。3月24日的深夜，北京知名画家大车却因创作有关
  <strong>
   武汉肺炎
  </strong>
  疫情的漫画《
  <strong>
   甩锅
  </strong>
  》，遭北京公安带走威胁并训诫了。
  <br>
   <br>
    一位艺术家创做个画有必要这么兴师动众？因为中共爱甩锅，所以害怕说。
    <br>
     <br>
      中共在疫情完全失控、扩散全世界之后，为逃脱罪责，先是内部各级官员互相甩锅、推卸责任。更怕被各国追究责任，不仅高举被收买的世卫组织（WHO）的说法，还将舆论矛头指向美国与欧洲重灾国，借此淡化最初隐瞒疫情的罪责。
      <br>
       <br>
        <strong>
         ●甩锅给美国、意大利、德国
        </strong>
        <br>
         <br>
          源自武汉的肺炎疫情因中共掩盖而酿成全球性的瘟疫。中共摆脱不了外界的质疑，于是开始制造舆论，炒作病毒来自美国。中共甩锅惹怒了美国政府，3月17号，总统
          <strong>
           <span href="https://www.secretchina.com/news/gb/tag/川普" target="_blank">
            川普
           </span>
          </strong>
          更直接了当的说，病毒来源就是在中国，就叫“中国病毒”。
          <br>
           <br>
            继美国之后，中共刻意扭曲“意大利疫情可能早在10月底或11月就已爆发”，暗示病毒是从意大利传到中国。
            <br/>
            <br/>
            接着，在大陆网站上又出现甩锅给德国的文章。版本是：默克尔2019年9月到访武汉，下榻宾馆距离华南海鲜市场只有200米，而德国的一间P4实验室2018年7月才开张，屡屡发生神秘事件，结合来看，是默克尔团队把病毒带到武汉。
            <br/>
            <br/>
            <strong>
             ●看看中共的甩锅历史
            </strong>
            <br/>
            <br/>
            <strong>
             一、甩锅给老天爷 大饥荒是人祸
            </strong>
            <br/>
            <br/>
            所谓“三年自然灾害”是指本世纪50年代末到60年代初，在中国大地上饿死数千万人。其实那几年既没有出现大规模的严重自然灾害，也没有发生大的战乱和严重瘟疫，而是天下太平，风调雨顺。
            <br/>
            <br/>
            为什么会发生1960年的大饥荒？主要是由于毛泽东亲自制定和强力推行的四项恶政造成的恶果。
            <br/>
            <br/>
            1、浮夸风导致的高指标、高征购，将农村生产队的粮食收缴一空
            <span id="hideid" name="hideid" style="color:red;display:none;">
             <span href="https://www.secretchina.com">
             </span>
            </span>
           </br>
          </br>
         </br>

       </br>
      </br>
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  2、“反瞒产私分”运动把农民家里收藏的少量救命粮搜刮得干干净净
 </p>
 <p>
  3、人民公社“大食堂”使生产队干部掌握了农民的生死权
 </p>
 <p>
  4、野蛮的“拦外流”政策剥夺了农民外出逃荒要饭的权利
  <br/>
  <br/>
  中共对这段它一手制造的历史大惨剧，以谎言“三年自然灾害”，把罪责推给老天爷。长期被中共欺骗的国人，特别是年轻后代们长期以来误以为“三年自然灾害”是导致大饥荒的原因。
 </p>
 <p>
  学者们在揭开历史真相的同时则呐喊：不能再冤枉老天爷了！
  <br/>
  <br/>
  <strong>
   二、把文革罪责推给帮凶 脱罪元凶
  </strong>
  <br/>
  <br/>
  “大饥荒”是毛时代的反右、大跃进、反右倾等一系列极左行为的恶果，又是
  <span href="https://www.secretchina.com/news/gb/tag/毛泽东" target="_blank">
   毛泽东
  </span>
  发动文革的根本起因。 毛泽东发动文革的真实动机，就是为了避免刘少奇等人在他死后清算他制造大饥荒的罪行。
  <br/>
  <br/>
  中共官方宣称“文化大革命”是毛泽东亲自发动和领导的，可是否定文革以后，中共只找了文革的打手、帮凶当替罪羊。
  <br/>
  <br/>
  文革期间，原南京军区某军副政委刘传新追随中共当局、成为执行上级命令的打手，将千余人列为“特务、叛徒、三反分子”，升得很快，官至北京市委常委、北京市政法委书记，北京市公安局局长。
  <br/>
  <br/>
  1977年1月27日，刘传新被免去市公安局长的职务，接受审查。5月19日，他自杀了。他下属的班子成员，也被“一锅端”，或隔离审查或停职检查。
 </p>
 <p>
  1977年11月，刘传新时期安排的793名军队干部全部撤离北京市公安局。其中17人被秘密枪决，没有经过公开审判程序。据知情者称，军队也按同样模式进行内部清理，把这批军人押解到云南秘密处决，并以“因公殉职”通知家属。
  <br/>
  <br/>
  中共还找了文革帮凶王、张、江、姚（王洪文、张春桥、江青、姚文元）四个替死鬼来承担文革的罪责，而真正的元凶即四人帮的主子中共党魁毛泽东却不予追究，因为再追究下去，祸源就是中共自己。
  <br/>
  <br/>
  <strong>
   三、导演“天安门自焚” 栽赃
   <span href="https://www.secretchina.com/news/gb/tag/法轮功" target="_blank">
    法轮功
   </span>
  </strong>
  <br/>
  <br/>
  2001年1月23日下午，天安门广场燃起5道火光。中共官媒宣称，5名“自焚者”都是法轮功学员。然而19年来，重重疑点不断曝光。
 </p>
 <p>
  1、刘春玲，中共宣称她自焚时当场死亡。然而将央视《焦点访谈》画面放慢，看到她被一名男子用物体重重击中头部而倒下。
 </p>
 <p>
  2、刘春玲的女儿刘思影，积水潭医院声称，为她做了气管切开术，但4天后，她接受央视采访时就能说话唱歌，被指有违医学常识。
 </p>
 <p>
  3、王进东，腿上的雪碧瓶烧不化；他身后的警察手持所谓灭火毯却不灭火，直到王进东喊口号后，才将他盖住。
 </p>
 <p>
  4、央视的画面，有远景、中景、近景。是谁在短短几分钟内，有备而来，不受阻拦，拍下这些画面？
 </p>
 <p>
  5、事件发生后，中共官媒新华社在2小时内，就迅速向外媒发布录像。《华尔街日报》记者张彦认为，这不符合通常中共新闻审查所需的时间。
 </p>
 <p>
  6、凡看过法轮功书籍的都知道，法轮功是禁止杀生和自杀的。一个真正的法轮功修炼者会做违背信仰的事吗？换而言之，违背信仰的能是个真正的修行者吗？
  <br/>
  <br/>
  如今，国际教育发展组织、中华民国国会等第三方观察，都认定这起事件是由中共一手炮制，意图挑起民众对法轮功的仇恨。
  <br/>
  <br/>
  <strong>
   ●甩锅弹回被扣 坐实“中共病毒”
  </strong>
  <br/>
  <br/>
  北京公安在暗夜里带走画家大车的原因 ，“就是漫画《甩锅》”（大车原话） 。中共为什么对甩锅这么敏感，就是因为过去也甩锅，现在也甩锅，本性喜甩锅，又不让人说。
  <br/>
  <br/>
  中共在引发这场祸及全球的大瘟疫之后，“
  <span href="https://www.secretchina.com/news/gb/tag/武汉肺炎" target="_blank">
   武汉肺炎
  </span>
  ”不但没有甩出，还成了川普口中的“中国肺炎”，但海内外的明智之士认为：导致武汉肺炎爆发的病毒是来自中共统治下的中国，由于中共当局隐瞒真相致使疫情在全球扩散。武汉人、湖北人，乃至所有中国人及全世界人民都是受害者。中共不是中国，也代表不了中国，因此，中共治下出现的这种病毒应叫“
  <strong>
   中共病毒

  称“武汉肺炎”为“中共病毒”，还有三个说法。

  <br/>
  <strong>
   一、中共从源头上就是邪的
  </strong>
 </p>
 <p>

 </p>
 <p>
  3、它是在“国外敌对势力”——苏联共产党的操控下建立起来的。
  <br/>
  <br/>
  <strong>
   二、
   <span href="https://www.secretchina.com/news/gb/tag/中共病毒" target="_blank">
    中共病毒
   </span>
   (又称新冠状病毒，COVID-19)的起源与流传过程
  </strong>
 </p>
 <p>
  1、病毒起源于德国（马克思），中间宿主是俄国（苏联共产党），经北大图书馆泄露，爆发于上海（1921年中共成立于上海）。
 </p>
 <p>
  2、此病毒在井冈山、延安多次变异，最终虐于神州大地，七十年无特效药根治，邓小平采取开放疗法，虽然疫情暂缓，但毛病未改，积恶成习。
 </p>
 <p>
  3、病毒基因序列经过人工重组，再次引发更大面积疫情，祸害全中国，危及全世界。
  <br/>
  <br/>
  <strong>
   三、 病毒就是针对共产党而来
  </strong>
  <br/>
  <br/>
  病毒严重的地区都是与中共联系紧密的地区，网上流传一幅地图，这个一带一路的地图，和武汉肺炎发病的地图是高度重合的。
  <br/>
  <br/>
  因为这次大瘟疫始发于中国武汉，人们就称它为“武汉肺炎”；又因为以上种种，人们又称之为“中共病毒”。
