 <p>
  相信许多使用者都习惯听从作业系统或软件的提示，进行更新。但
  <span href="https://www.secretchina.com/news/gb/tag/Windows" target="_blank">
   Windows
  </span>
  10 的使用者要注意了，5月的更新档别急着下载更新，以免破坏 Chrome 安全机制。
  <span id="hideid" name="hideid" style="color:red;display:none;">
   <span href="https://www.secretchina.com">
   </span>
  </span>
 </p>
 <p>
  根据外媒《SlashGear》报导内容指出，
  <strong>
   微软
  </strong>
  （
  <span href="https://www.google.com.tw/url?sa=t&amp;rct=j&amp;q=&amp;esrc=s&amp;source=web&amp;cd=17&amp;cad=rja&amp;uact=8&amp;ved=2ahUKEwjxgsW8xYvpAhUSyYsBHQA_AjwQmhMwEHoECBgQGA&amp;url=https%3A%2F%2Fzh.wikipedia.org%2Fzh-tw%2F%25E5%25BE%25AE%25E8%25BD%25AF&amp;usg=AOvVaw1bMUCS0QZzEDS7AWLFecO3">
   Microsoft
  </span>
  ）在
  <span href="https://www.google.com.tw/url?sa=t&amp;rct=j&amp;q=&amp;esrc=s&amp;source=web&amp;cd=21&amp;cad=rja&amp;uact=8&amp;ved=2ahUKEwjF9-fVxIvpAhV7xYsBHXBIAdQQmhMwFHoECBUQGQ&amp;url=https%3A%2F%2Fzh.wikipedia.org%2Fzh-tw%2FWindows_10&amp;usg=AOvVaw1UQ6bSIXY7Q_Ng6vL7DkX7">
   Windows 10
  </span>
  1903 更新档又被发现一项 Bug，可能造成包括
  <span href="https://www.google.com.tw/url?sa=t&amp;rct=j&amp;q=&amp;esrc=s&amp;source=web&amp;cd=17&amp;cad=rja&amp;uact=8&amp;ved=2ahUKEwjLv_rlxIvpAhUNCpQKHV4GC6gQmhMwEHoECBMQGA&amp;url=https%3A%2F%2Fzh.wikipedia.org%2Fzh-tw%2FGoogle&amp;usg=AOvVaw0OTulaovKrZ-b7Y12gM-la">
   Google
  </span>
  <span href="https://www.google.com.tw/url?sa=t&amp;rct=j&amp;q=&amp;esrc=s&amp;source=web&amp;cd=14&amp;cad=rja&amp;uact=8&amp;ved=2ahUKEwia84TvxIvpAhWoy4sBHRcjCMoQmhMwDXoECBMQGA&amp;url=https%3A%2F%2Fzh.wikipedia.org%2Fzh-tw%2FGoogle_Chrome&amp;usg=AOvVaw0_J2OocByB644xi5rXpZB9">
   Chrome
  </span>
  、微软
  <span href="https://www.google.com.tw/url?sa=t&amp;rct=j&amp;q=&amp;esrc=s&amp;source=web&amp;cd=32&amp;cad=rja&amp;uact=8&amp;ved=2ahUKEwihloP6xIvpAhUoCqYKHW-eBEMQmhMwH3oECBEQGQ&amp;url=https%3A%2F%2Fzh.wikipedia.org%2Fzh-tw%2FMicrosoft_Edge&amp;usg=AOvVaw0JqAPpniIZz7jHtfeuol_7">
   Edge
  </span>
  在内，基于 Chromium 开源架构的浏览器失去 Sandbox（
  <strong>
   沙盒
  </strong>
  ）的安全防护。这也使浏览器被网络攻击的机率相对提高。
 </p>
 <p>
  Sandbox（
  <span href="https://www.secretchina.com/news/gb/tag/沙盒" target="_blank">
   沙盒
  </span>
  ）是 Google 在 Chromium 架构中专门设置来保护用户安全的技术，能够保护浏览器的核心，避免未授权的程式取用核心资源。话虽如此，这项技术最关键的保护部份，还是要看基于何种作业系统而定。
 </p>
 <p>
  只是在微软针对 2020 年 5 月释出的 Windows
  <span href="https://www.secretchina.com/news/gb/tag/10" target="_blank">
   10
  </span>
  1903 更新档案中，却被发现对 Google 的 ZeroDay（零日漏洞）计划中的关键代码进行修改。这个动作很可能造成 Windows 10 上的 Chromium 架构浏览器将因此失去 Sandbox（沙盒）的安全防护。
 </p>
 <p>
  由于许多浏览器是基于 Chromium 架构开发出来的，就包括微软自己推出的 Edge 浏览器、
  <span href="https://www.google.com.tw/url?sa=t&amp;rct=j&amp;q=&amp;esrc=s&amp;source=web&amp;cd=27&amp;cad=rja&amp;uact=8&amp;ved=2ahUKEwiJgO2SxYvpAhUMkJQKHaUrA3QQmhMwGnoECBMQGQ&amp;url=https%3A%2F%2Fzh.wikipedia.org%2Fzh-tw%2FOpera%25E9%259B%25BB%25E8%2585%25A6%25E7%2580%258F%25E8%25A6%25BD%25E5%2599%25A8&amp;usg=AOvVaw2HS4f6NzRbGZW8lWEOSKHm">
   Opera
  </span>
  、
  <span href="https://www.google.com.tw/url?sa=t&amp;rct=j&amp;q=&amp;esrc=s&amp;source=web&amp;cd=30&amp;cad=rja&amp;uact=8&amp;ved=2ahUKEwjrsp-hxYvpAhWnHqYKHYG7AeoQmhMwHXoECBQQGQ&amp;url=https%3A%2F%2Fzh.wikipedia.org%2Fzh-tw%2FBrave%25E6%25B5%258F%25E8%25A7%2588%25E5%2599%25A8&amp;usg=AOvVaw2lGVfl-ur0YOcT3L1qY3dz">
   Brave
  </span>
  ，以及被赞誉多年，认为安全性很高的 Mozilla
  <span href="https://www.google.com.tw/url?sa=t&amp;rct=j&amp;q=&amp;esrc=s&amp;source=web&amp;cd=18&amp;cad=rja&amp;uact=8&amp;ved=2ahUKEwjoq46txYvpAhWpw4sBHbShCQkQmhMwEXoECBQQGQ&amp;url=https%3A%2F%2Fzh.wikipedia.org%2Fzh-tw%2FFirefox%25E7%2580%258F%25E8%25A6%25BD%25E5%2599%25A8&amp;usg=AOvVaw3S3bjRvPpDUnxxmLO3q7dy">
   Firefox
  </span>
  浏览器，也因为此次的更新，连带受到影响。
 </p>
 <p>
  目前微软官方尚未针对此事进行回应，且根据安全人员的分析，若如果没有其他系统级的重大漏洞，骇客尚无法直接利用这次的 Sandbox（沙盒）漏洞。只是这并不意味此次的 Bug 不会带来其它后续的影响。所以还是建议用户在微软还没调整
  <span href="https://www.secretchina.com/news/gb/tag/更新" target="_blank">
   更新
  </span>
  档之前，不妨先暂停升级到最新版本的 Windows 系统。
