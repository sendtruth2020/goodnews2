  【看中国2020年6月20日讯】今天来谈一个有关“三海战略”的话题。
  <br>
   <br>
    所谓“三海”，是指东海、台海与南海，也是美中两国地缘战略博弈最为集中的区域。它以什么方式呈现，目前已经出现一个大致轮廓，这种局面可能要持续相当一段时间。
    <br>
     <br>
      <strong>
       <span href="https://www.secretchina.com/news/gb/tag/台湾" target="_blank">
        台湾
       </span>
       防卫法
      </strong>
      <br>
       <br>
        首先来看美国的最新动向。美国联邦参议员霍利（Josh Hawley）在官方推特上表示，他在6月10日提出《台湾防卫法》（Taiwan Defense Act）草案，目的是要确保美国有足够的军事能力，遏阻中国对台湾所采取的“既成事实”（fait accompli）。
        <br>
         <br>
          根据草案的定义，所谓“既成事实”，是指中国意图在美军做出有效反应前，以军事手段夺取对台湾的控制权，同时威慑美军，使美国相信采取反制行动将非常困难，或者要付出高昂代价。为此，草案要求美军建构具有阻滞（delay）、降级（degrade），最终能够击败（defeat）中国企图进犯台湾的能力。
          <br/>
          <br/>
          估计，美国通过《台湾防卫法》的难度，比通过《台湾旅行法》（Taiwan Travel Act）和《台北法案》（Taipei Act）大得多。因为它碰触美中关系中最敏感的神经，也是台美关系从量变到质变最具有指标性的行动之一。由此，美军介入台海冲突开始有了最直接的法律依据。
          <br/>
          <br/>
          不管《台湾防卫法》通过与否，防卫台湾的议题已经在美国政界引起高度重视，美国军方在国会议员的敦促下也开始要正视防卫台湾的可能性，而有所评估和准备。

          <br/>

  美国国防部长埃斯珀（Mark Esper）6月15日投书媒体，说明美国推动“印太战略”的最新进展。显示在“三海”的前沿部署极为重要。（图片来源：MANDEL NGAN/AFP via Getty Images）
 </p>
 <p>
  除了霍利的提案，美国国防部长埃斯珀（Mark Esper）6月15日以〈美国与伙伴们并肩推动自由开放的印太区域〉为题，投书新加坡《海峡时报》（The Straits Times）。他从战备整备、加强伙伴关系及促进区域网络化等三方面，说明美国推动“印太战略”的最新进展。
  <br/>
  <br/>
  埃斯珀在文章中说：美军持续透过安全合作、信息共享及联合演习，与印太地区的盟友及伙伴建立更紧密的关系。他还说，美国维持对民主台湾的承诺。请注意，埃斯珀提到的盟友及伙伴国家，有一些处于
  <span href="https://www.secretchina.com" target="_blank">
   中国
  </span>
  的前沿地带，如日本、台湾和菲律宾，沿着东海、台海及南海一字排开；另有一些则在中国的侧翼，如印度与东盟国家；还有一些在前沿地带的外围，如澳洲、新西兰及多个南太平洋岛国。摊开地图，其重心就在“三海”。
  <br/>
  <br/>
  反观中国，从1990年代初起，她的“积极防御”战略从本土防卫走向外向型的战略布势，向周边地区乃至于全球扩张其战略利益，逐步推进战略前沿，拓展战略纵深，并且聚焦在“打赢主要来自海上方向的信息化局部战争”。所谓的“海上方向”，主要是指东海、台海和南海的方向，其重心也在“三海”。
  <br/>
  <br/>
  两相对照。可以说，美中两国地缘战略博弈在今后一段时间内，将沿着“三海”及其周边海域而展开。
  <br/>
  <br/>
  <strong>
   三海联动
  </strong>
 </p>
 <p>
  中国的“三海”战略强调分而治之，各个击破。图为中国辽宁舰航母编队在巡航。（美联社）
 </p>
 <p>
  美国对“三海”的关注，是根据中国向外扩张的时间顺序而做出回应。首先是在东海。2013年11月，中国从宣布划定并施行东海防空识别区（ADIZ）起，战机开始频繁到东海实施战备巡航。根据日本防卫省的统计，2019年度，日本航空自卫队战机紧急升空的次数达947次，其中71%是为了应对中国战机，而应对来自东海方向的中国战机就多达61%。
  <br/>
  <br/>
  这不是最近一年发生的事。从2015年度到2019年度，日本战机紧急升空应对中国战机的次数，占总数的比例少则55%，多则高达73%，其中应对来自东海方向的中国战机，最高达到69%。换言之，日本战机紧急升空的主要压力来自东海。这还不包括日本和中国执法舰船在东海的紧张对峙。为此，美国进一步强化与日本的防务合作。
  <br/>
  <br/>
  紧接着是南海。从2013 年起，中国开始在南沙岛礁展开大规模的填海造陆，至今在永暑、渚碧、美济等3岛各兴建3,000米的飞机跑道，并且部署鹰击-12B反舰导弹与红旗-9B远程防空导弹，岛礁军事化的程度大幅增强，逐步翻转周边海域的战略态势。为此，美军以“航行自由行动”加强在南海的巡航，确保区域的稳定。
  <br/>
  <br/>
  根据公开的资料显示，2019年美国海军舰船进入中国宣称拥有12海里领海的岛礁范围有8次，比上年增加60%，而且覆盖中国在南沙、西沙和中沙群岛等相关岛礁。从执行任务的舰船来看，除了少数来自部署在日本的第七舰队，其余大多来自以美国西海岸为基地的第三舰队，说明第三舰队从东太平洋前推到西太平洋，加强与第七舰队进行更紧密的“前沿作战”（forward operating）合作，应对中国在东海与南海的扩张，其构想从2015年9月提出到今天，已经充分落实。今后还将进一步强化。
  <br/>
  <br/>
  在台海方面，2016年5月，蔡英文总统正式就职，民进党在立法院取得过半席次，首度完全执政，并且不承认北京提出的“九二共识”和“一国两制”，两岸关系交恶。中国军机从这一年的11月起，开始绕台飞行，陆续在台湾周边海域进行海空联合演练，检验战区多军种一体化联合作战能力，对区域及台湾的安全环境带来紧迫性的威胁。为此，美国开始重视与台湾的合作，先后通过《台湾旅行法》和《台北法案》，同时美军加大在台海附近巡航的频率与力度。
  <br/>
  <br/>
  美国联邦参议员霍利上周提出《台湾防卫法》草案，可以说是美国政界对协防台湾最直接而又坚决的一次宣示，也预告在美国“印太战略”的框架下，位于中国大陆最前沿的台湾，所具有的特殊地位和关键性角色。
  <br/>
  <br/>
  <strong>
   前沿部署
  </strong>

 <p>
  从霍利提出的《台湾防卫法》草案，以及埃斯珀在新加坡《海峡时报》投书的内容来看，他们并没有具体提到“三海”，但是在字里行间，都在强调美军在前沿部署的重要。埃斯珀特别指出，美军正在发展21世纪新的联合作战准则，包括实行“分散式海上作战”（Distributed Maritime Operations）等新的作战概念。这个概念将彻底颠覆美国海军在前沿部署及兵力运用的惯例，以一种新的作战方式，应对中国的挑战。
  <br/>
  <br/>
  二战后，美国海军作战以航母打击群为核心，在全球巡航执行战备，可谓无往不利。但是这种集中式的作战模式，一旦面对中国东风系列反舰导弹饱和攻击时，反而陷入危险。因此，美国海军把航母打击群化整为零，并将非航母编队的舰船进行重组，以分散式的海上作战模式，保持海上优势。
  <br/>
  <br/>
  它的具体作法是，以小型的水面舰艇编队和无人舰船进行前沿部署，作为主要吓阻及作战主力，而传统航母打击群将保持在敌方反舰导弹打击范围之外的相对安全海域，为前沿编队提供监侦、预警和电子战支援。这样既可以保持在前沿作战的灵活度，又能避开大型舰船过于前出而遭到攻击的危险。
  <br/>
  <br/>
  换言之，今后美军一旦协防台湾，航母打击群将不会暴露在前沿地带，而是由更多的小型舰船担任第一线的主攻，不但可以大幅降低被侦获的机率，还能够藉由数据链路，分进合击，完成“分散式杀伤”的要求。

  除了战术上的战备整备。从战略的角度看，埃斯珀提到加强伙伴关系及促进区域网络化，是美国推动“印太战略”中的另外两大支柱，显示在前沿地带阻断中国的军力扩张，需要整合盟邦和伙伴之间的力量。换言之，在“三海”展开军事行动，并非分别进行，而是互为联动。一旦有事，可以相互策应，使中国难以同时应付。
  <br/>
  <br/>
  从中国的角度看，她的“积极防御”战略从本土防卫走上外向型战略。在东海方向，强调管控，守住东海防空识别区，力保东南沿海经济区不受外敌侵入；在南海方向，重在维权，以岛礁军事化声索南海主权，同时确保海上运输线，不被外力遮断；在台海方向，则是争取战略主动权，能够封控台海，前出岛链，拓展战略纵深，阻滞美军介入台海冲突。趁早做好战场经营，避免战略被动，为日后两岸统一创造有利条件。
  <br/>
  <br/>
  至今，解放军虽没有“三海战略”的提法，却有实质内涵。按北京设想，“三海”相互联动，防卫上则各司其职，最好能分而治之，避免多国同时反制。如东海方向有美日安保、台海方向有台湾和美国、南海方向有东盟国家，尤其是越南和菲律宾。“三海”一旦同时有事，任凭北京有三头六臂，也难以应付。因此，要分而治之，各个击破。这和美国的设想：“三海”相互联动，分进合击，截然不同。
  <br/>
  <br/>
  美中双方的角力，历史将会在可期的未来给出答案。
